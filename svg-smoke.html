SVG smoke test — Chrome built-in AI Prompt API (Gemini Nano)

Asks the on-device Prompt API to return an SVG from a short prompt, then shows the RAW model output and the strict validation result. Browser-only: requires a Chrome build with the Prompt API / Gemini Nano enabled.

Open this page through a Vite server, not as a file:// — the TypeScript module below only loads when served. Dev: npm run dev → http://localhost:5173/svg-smoke.html. Preview: npm run build && npm run preview → http://localhost:4173/svg-smoke.html.
